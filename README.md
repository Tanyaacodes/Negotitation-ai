🤝 Negotiation AI: The Ultimate Deal-Maker Challenge
Negotiation AI ek immersive negotiation game hai jahan aapka muqabla ek AI Seller se hota hai. Aapka mission hai ek fixed product ko sabse kam dam (lowest possible price) mein kharidna. Ye sirf ek chat app nahi hai, balki aapki persuasion skills, logic, aur patience ka test hai.

Powered by Google Gemini API, ye AI seller apne hidden constraints (Minimum Price, Target Profit) ko protect karta hai, aur aapko use convince karna hota hai deal final karne ke liye.

🚀 Key Features
Gemini AI Integration: Real-time, intelligent aur adaptive responses jo har negotiation ko unique banati hain.

Dual Difficulty Modes:

🟢 Noob Mode: AI thoda "soft" hai. Ise emotional baaton se exploit aur manipulate kiya ja sakta hai. Ye jaldi price drop kar deta hai.
🔴 Pro Mode: AI ek seasoned negotiator hai. Ise manipulate karna namumkin hai. Ye aapke har argument ko logic se counter karega aur apni "Bottom Line" ko sakhti se follow karega.

Global Leaderboard: Aapka final deal price decide karega ki aap duniya ke baaki negotiators ke muqable kahan stand karte hain.

🛠️ Tech Stack
Frontend: React.js / Next.js
Backend: Node.js (Deployed on Render/Vercel)
AI Engine: Google Gemini Pro API

Styling: Tailwind CSS / CSS Modules
